# Joris Suarez - Portfolio

Personal portfolio website showcasing my work as a Web Developer & UI Designer.

## Features

- **Bilingual Support** - Full French/English language switching with persistent preferences
- **Glassmorphism Design** - Modern iOS-inspired UI with animated gradients and blur effects
- **Responsive Layout** - Optimized for all screen sizes
- **Multiple Pages** - Home, About, Services, Works, Blog, and Contact sections

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox/Grid, Animations)
- Vanilla JavaScript
- Google Fonts (Inter)

## Project Structure

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

No build tools or dependencies required.
